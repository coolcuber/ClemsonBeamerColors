This is just a beamercolors file that is made to use Clemson colors in the `Madrid` theme.  This might
look alright with other themes as well, but I haven't tried it.  Here is a sample slide that uses
this color theme:

![slide](images/sample.png)
To use this color scheme, just put `beamercolorthemetiger.sty` in the same directory as your main
file and add the lines
```
\usetheme{Madrid}
\usecolortheme{tiger}
```
to your main file.  If you like it or think any changes should be made, let me know!
